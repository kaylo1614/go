# ⚫⚪ 烏鷺爭霸 (CrowHeronGo)

> **結合傳統圍棋哲學與現代卡牌策略的創新對弈遊戲。**
> *A strategic fusion of traditional Go (Weiqi) and deck-building mechanics.*

## 🎮 遊戲簡介 (Introduction)

《烏鷺爭霸》是一款在 9x9 棋盤上進行的策略遊戲。不同於傳統圍棋，玩家不僅可以單子落子，更能透過消耗「能量 (Mana)」發動卡牌，一次性佈下特定的棋型（如虎口、小飛、扭斷）。

遊戲的核心在於資源管理：**要在何時「單子落子」來累積資源？又要在何時「打出卡牌」來搶佔地盤？**

### ✨ 特色
* **非對稱牌庫**：黑方擁有攻擊型專屬卡，白方擁有靈活型專屬卡。
* **幾何變換引擎**：卡牌支援旋轉 (0°/90°/180°/270°) 與鏡像翻轉。
* **資源交易系統**：獨特的「棄牌換資源」機制，讓手牌管理成為勝負關鍵。
* **跨平台遊玩**：支援 Windows 執行檔與 Web (iPhone/Android) 網頁版。

## 🕹️ 遊戲規則 (Game Rules)

### 1. 獲勝條件
採用 **中國數子法 (Area Scoring)** 結算：
* **子空皆地**：盤面上的活棋 + 圍住的空地 = 總分。
* **公氣判定**：若空地同時接觸黑白雙方，雙方各得 0.5 分。
* **勝負判定**：
    * 白方獲得 **7.5 目** 的貼目補償。
    * **黑方總分 ≥ 44** 即獲勝，否則白方勝。

### 2. 資源系統
* **寶石 (Mana)**：發動卡牌的能量。初始 1 顆，上限 3 顆。
* **手牌 (Hand)**：初始 3 張，上限 5 張。

### 3. 回合流程 (Turn Structure)
輪到玩家時，回合分為兩個階段：**「行動階段」** 與 **「交易/結束階段」**。

#### **第一階段：行動 (Action)**
玩家必須從以下三者擇一進行：
* **A. 單子落子 (蓄力)**：
    * 在棋盤下 1 顆子。
    * **獎勵**：獲得 1 顆寶石 + 抽 1 張牌。
* **B. 卡牌落子 (進攻)**：
    * 消耗卡牌標示的寶石數量。
    * 選擇錨點與方向，一次性落下多顆棋子。
* **C. 虛手 (Pass)**：
    * 放棄行動，直接結束回合（不獲得資源）。

#### **第二階段：交易 (Trade) - *僅在行動後可用***
在執行完「落子」或「卡牌」後，可進行資源交換：
* **換能量**：棄掉 2 張手牌 -> 獲得 1 顆寶石。
* **換牌**：棄掉 2 張手牌 -> 從牌庫抽 1 張牌。

## 🖱️ 操作指南 (Controls)

| 動作 | 操作方式 | 說明 |
| :--- | :--- | :--- |
| **落子/選牌** | **左鍵點擊** / **單指觸碰** | 選擇卡牌、放置棋子、點擊按鈕。 |
| **取消/悔棋** | **右鍵點擊** / **ESC 鍵** | 取消當前選取的卡牌，回到單子模式。 |
| **全螢幕** | **F11 鍵** | 切換視窗/全螢幕模式 (僅限桌面版)。 |
| **結算** | 點擊「結算」按鈕 | 進入死子清理模式，標記死棋後計算分數。 |

## 🚀 如何遊玩 (How to Play)

### 🌐 網頁版 (iPhone/Android/PC)
1. 確保已部署至 GitHub Pages。
2. 使用瀏覽器開啟專案網址即可遊玩。

### 💻 本地版 (Windows)
1. 前往 `dist` 資料夾下載 `CrowHeronGo.exe`。
2. 雙擊執行即可。
